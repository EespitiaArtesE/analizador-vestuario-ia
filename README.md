# Analizador de vestuario para Artes Escénicas

##Descripción del proyecto
Sistema de asistencia de inteligencia artificial para el diseño de vestuario para artes escénicas, deasrrollado con metodología ágil y herramnientas de aprendizaje profundo.

## Estructura proyecto
- `/dataset`-Fichas técnicas y datos de entrenamiento
- `/backend`-API u lógica de servidor
- `/frontend`- Interfaz de ususario
- `/modelos`- Modelos de machine learning entrenados
- `/documentacion`-Metodología, procesos y artefactos ágiles
- `/notebooks`- Notebooks de Google colab para enterenamiento
