# User Stories Priorizadas

### Prioridad alta (Sprint 1-2)
1. **US-01:** Como clienta, quiero ingresar la información de mi proyecto de vestuario a traves de un formulairo web para que el sistema pueda analizarlo.
   
2. **US-02:** Como sistema quiero almacenar la información del cliente para poder procesarla y generar sugerencias.
   
3. **US-03:** Como desarrolladora, quiero un data set estructurado  con mis fichas técnicas para entrenar el modelo de IA.

4. **US-04:** Como esperta en vestuario quiero que el sistema respete las consideraciones culturales y patrimoniales en sus análisis.

### Prioridad media (Sprint 3)

5. **US-05:** Como sistema, quiero analizar las imagenes de referencia usando vision por computador para identificar elementos culturales clave.

6. **US-06:** Como cienta, quiero recibir un mood board visual con referencias para validar la dirección del diseño.

7. **US-07:** Como experta en vestuario quiero que el diseño que el sistema demuetre capacidades de aprendizaje profundo con redes neuronales.
### Prioridad media baja (Sprint 4)

8. **US-08:** Como usuario, quiero poder conversar con un chatbot que me explique el proceso y las sugerencias  del sistema.
9. **US-09:** Como proyecto académico queiro estar completamente documentado sobre la realización de la herramienta.
10. **US-10:** como profesional, quiero poder presentar habilidades técnicas.
